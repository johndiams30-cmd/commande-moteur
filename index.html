📖 À propos du projet

🚀 IoT 📡 ESP32 💡 ThingSpeak 🎨 Chart.js

Ce projet a été mis en place par Ir DIAMONEKA VAKANDA Jonathan pour contrôler à distance un moteur connecté sur internet via ThingSpeak.

✨ Fonctionnalités

  • 🔌 Contrôle à distance du moteur (Marche/Arrêt)
  • 📊 Visualisation de l'historique des commandes en temps réel
  • 📈 Graphique d'évolution de l'état du moteur
  • 🌓 Thème clair/sombre personnalisable
  • 📱 Design responsive (mobile, tablette, desktop)
  • 🔐 Accès sécurisé par mot de passe

🛠️ Technologies utilisées

  • 🎯 Frontend : HTML5, CSS3, JavaScript
  • 📊 Graphiques : Chart.js
  • ☁️ Cloud : ThingSpeak API
  • 🔌 Microcontrôleur : ESP32

📡 Communication

L'ESP32 lit les commandes depuis ThingSpeak et agit sur le moteur. L'interface web envoie les commandes via l'API ThingSpeak et affiche l'état en temps réel.
